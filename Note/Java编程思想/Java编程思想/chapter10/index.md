#第十章 内部类

**可以将一个类的定义放在另一个类的定义内部，这就是内部类。**

内部类允许你把一些逻辑相关的类组织在一起，并控制位于内部的类的可视性。

特点：

1. 内部类看起来像是一种代码隐藏机制：将类置于其他类的内部
2. 内部类了解外围类，并能与之通信。
3. 内部类代码优雅而清晰
